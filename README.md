# CAEFISS 2.0 Development Benchmarker

## Overview

This tool is built for the Canadian Adverse Events Following Immunization Surveillance System (CAEFISS) to provide advanced analytics of development performance.

## Usage

To use the tool, export the audit log JSON file from the Jira > Automation > done-tracker trigger.

Then, upload the JSON file using the file input and generate your choice of metrics.

Once generated, you can simply download the image.

# Technology

This application was built using Vite + React and uses a Python FastAPI
